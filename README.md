# portfolio_V3-front-and-back-
Portfolio website V3. Mostly JavaScript oriented.

React 18 + TypeScript
├── Vite (Build Tool)
├── Tailwind CSS (Styling)
├── Framer Motion (Animations)
├── React Query (Data Fetching)
├── React Router (Navigation)
├── Axios (HTTP Client)
└── Lucide React (Icons)


Node.js + TypeScript + Express
├── Prisma (ORM)
├── PostgreSQL (Database)
├── JWT (Authentication)
├── Nodemailer (Email)
├── Express Rate Limit
├── Helmet (Security)
└── Winston (Logging)
